Name of file: bikeshare.py

Description: This project analyzed data for renting of bikes in three cites (New york, Chicago and Washington)
             The analysis produced the following results:
                  Most common month, year and day of rentage
                  Most common start station
                  Most common end station
                  Most common journey
                  Total trip duration, Mean trip Duration
                  customer type counts
                  Gender count

Files Used: chicago.csv, new_york_city.csv, washington.csv
